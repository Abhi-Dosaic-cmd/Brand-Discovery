The form is integrated with Google Sheets via Google Apps Script.

### Setup Instructions (Do it yourself in 2 minutes):
1. Open the [submit_to_google_sheets.gs](./submit_to_google_sheets.gs) file.
2. Go to your Google Sheet > **Extensions** > **Apps Script**.
3. Paste the code and click **Deploy** > **New Deployment**.
4. Set access to **"Anyone"**.
5. Copy the URL and paste it into `script.js`.
